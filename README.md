# Open App Builder Server

**PreRequisites**

- Docker
- Node

## Local Development

```sh
cp .env.example .env
npm run dev
```

Access through https://localhost:8000
Use the `DASHBOARD_USERNAME` and `DASHBOARD_USERNAME` variables set in `.env`
